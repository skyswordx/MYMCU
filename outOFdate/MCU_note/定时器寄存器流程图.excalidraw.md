# Text Elements
定时器框图 ^DfoylCVU

STC89C52的T0和T1计时器均有四种工作模式：
	模式0：13位定时器/计数器
	模式1：16位定时器/计数器（常用）
	模式2：8位自动重装模式
	模式3：两个8位计数器
 ^0XLbSZh4

这个是timer high和timer low寄存器
分别是高八位和低八位寄存器，用来加1记数
可以记录从0到65535的数
 ^kPdVLQfo

当达到最大值65535之后再加1记数
timer high和timer low就会重新变成0
 ^BTSp04oI

然后timer flag这个溢位标志寄存器就会有记录
由此可以作为中断源向中断系统发出中断申请 ^oXJgiGdW

system clock
MCU内置的系统时钟,
它是从晶振中获取脉冲来进行计时的
一般晶振的表面会写它的频率 ^t96kbkHm

T0 Pin
一般被称作T0口
如果此时你需要接上外部是时钟源的时候, 就需要用到它
如果前面的开关指向这个口的话, 说明你正在使用外部的时钟源

t0口与sysclk可以共存
你想用哪个直接配置就可以了

如果使用外部的时钟
其实就是给外部时钟作为计数器了 ^xYiH6l6z

计数单元 ^vpUXKpvA

C/T代表counter or timer
如果其值为1，则是counter
如果其值为0，则是timer ^gYeIgrFR

时钟 ^8vw18Rsl

非门 ^59oovmBu

与门 ^qYIvxdvS

也就是说，要让计数单元运行 ^Rzm0wWTR

     还有另一端也给1 ^S0bBvBtG
